Detailed Description

Sudoku solver tool which calls from the src module.

This module takes in an input.txt file with the structure described in the READ.me file. It calls the input_converter function from the src module to convert the input to a numpy array. It calls the solve_sudoku_wrapper function from the src module to solve the sudoku puzzle using a backtracking algorithm. It then calls the output_converter function from the src module to convert the outputted numpy array back to the same format as the input.txt file. When called from the terminal, the function prints a solution to the inputted sudoku puzzle to the terminal.
