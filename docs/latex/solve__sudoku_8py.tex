\doxysubsection{Detailed Description}
This module contains a function and a wrapper function to find empty cells in a puzzle and implement a backtracking algorithm.

This module contains a wrapper function which makes a list of the indices of empty cells in a sudoku puzzle. The wrapper function then calls the solve\+\_\+sudoku function, which implements a backtracking algorithm using recursion to solve the sudoku puzzle.
